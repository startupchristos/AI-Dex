# Identity

Personal identity context for life-aware AI interactions. Dex and related skills (career-coach, deep-solve, first-principles) read these files when providing advice, planning, or decision support.

## Files

| File | Purpose |
|------|---------|
| `Beliefs.md` | Core beliefs and worldview |
| `Challenges.md` | Current obstacles and constraints |
| `Wisdom.md` | Collected insights, lessons, mental models |

## When to Update

- **Beliefs:** When you articulate what you stand for or a principle that guides you
- **Challenges:** When you name a blocker, pattern, or trade-off you're working through
- **Wisdom:** After significant work, reviews, or decisions — capture what you learned

## Integration

Skills that use identity context:
- `/career-coach` (Depth mode)
- `/deep-solve`
- `/first-principles-custom` — challenge assumptions against your beliefs
- `/council-custom` — multi-perspective decisions
